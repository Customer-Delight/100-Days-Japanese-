- [ ] Works on mobile phones (small screens)

Test on:

- [ ] Fast load times

## 🐛 Reporting Bugs

### How to Report
1. **Check existing issues** first to avoid duplicates
2. **[Create a new issue](https://github.com/Customer-Delight/100-Days-Japanese-/issues/new?template=bug_report.md)** using our Bug Report template
3. **Include as much detail as possible**:
   - Device and browser information
   - Clear steps to reproduce
   - Screenshots or videos
   - Console errors (F12 → Console tab)

### Bug Severity Levels
- **🔴 Blocking**: App crashes or core feature completely broken
- **🟠 Major**: Significant functionality broken, but app still usable
- **🟡 Minor**: Non-critical issue, workaround available

### Example Bug Report
```
Title: [BUG] Login button doesn't respond on iPhone 12

Environment: Safari on iOS 15.2, iPhone 12

Steps to Reproduce:
1. Open PWA on iPhone
2. Click the login button
3. Expected: Login screen appears
4. Actual: Nothing happens, button doesn't respond

Screenshot: [attached]
Console Error: "Click handler undefined"
```

## 📝 Issue Labels

We use the following labels to organize issues:

| Label | Color | Meaning |
|-------|-------|---------|
| `bug` | 🔴 Red | Bug report |
| `bug-blocking` | 🔴 Dark Red | Blocking bug - critical |
| `bug-major` | 🟠 Orange | Major bug |
| `bug-minor` | 🟡 Yellow | Minor bug |
| `enhancement` | 💙 Blue | Feature request |
| `help wanted` | 💚 Green | We need community help |
| `good first issue` | 💜 Purple | Good for newcomers |
| `needs-triage` | 🟠 Gold | Needs review |
| `question` | 💗 Pink | Question/discussion |
| `duplicate` | ⚫ Gray | Duplicate issue |
